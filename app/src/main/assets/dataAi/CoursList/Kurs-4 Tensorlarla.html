🎓 Tensorlarla İşləmə (PyTorch nümunəsi ilə):

Tensorlar sadə dillə desək, Numpy array-lərinə bənzəyirlər, amma onlar GPU-da işləyə bilir, bu da deep learning üçün çox vacibdir.

✅ Nümunə: Tensor yaratmaq

import torch

# 1D tensor
a = torch.tensor([1.0, 2.0, 3.0])
print(a)

# 2D tensor
b = torch.tensor([[1, 2], [3, 4]])
print(b)

# Zəruri hallarda Random tensor:
c = torch.rand((2, 3))  # 2x3 ölçülü təsadüfi ədədlərlə dolu tensor
print(c)

⚙ Tensorlarla riyazi əməliyyatlar:

x = torch.tensor([1.0, 2.0, 3.0])
y = torch.tensor([3.0, 2.0, 1.0])

print(x + y)         # Toplama
print(x * y)         # Hər elementə görə vurma
print(torch.dot(x, y))  # Skalyar hasil

🎯 Modelin Test Edilməsi

Öyrətdiyin modeli test etmək üçün sənə aşağıdakı kimi bir yanaşma lazımdır:

# model artıq öyrədilib və test mərhələsindəyik
model.eval()  # Modeli "eval" rejiminə keçirt

with torch.no_grad():  # Gradient hesablanmasın
    y_pred = model(X_test)
    predicted = torch.round(y_pred)  # Əgər Binary classification-dursa
    accuracy = (predicted == y_test).float().mean()
    print(f"Modelin düzgünlük faizi: {accuracy:.4f}")
