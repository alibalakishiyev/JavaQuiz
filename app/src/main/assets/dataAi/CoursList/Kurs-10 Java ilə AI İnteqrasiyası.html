5️⃣ Java ilə AI İnteqrasiyası – Plan

Burada məqsədimiz .pth və ya .h5 formatında təlim olunmuş AI modelini Java backend-ə qoşmaqdır. Aşağıdakı yollarla mümkündür:

🔌 Variant 1: Python Backend ilə Rest API yaratmaq

  • Python-da Flask və ya FastAPI ilə AI model yüklə (məsələn: torch.load()).
  • POST sorğusu ilə Java-dan şəkil/giriş verisini Python serverinə göndər.
  • Python modeli nəticəni çıxarır, Java onu alır və istifadə edir.

➡️ Bu üsul ən çox istifadə edilən yoldur – çünki PyTorch modelini birbaşa Java-da işlətmək çətindir.

🛠️ Variant 2: ONNX (Open Neural Network Exchange)

  • .pth modelini ONNX formatına çevir.
  • Java-da onnxruntime kitabxanası ilə modeli işlə.

Bash (Python) kodu:

# PyTorch modelini ONNX formatına çevirmək
torch.onnx.export(model, input_tensor, "model.onnx")

Java kodu:

OrtEnvironment env = OrtEnvironment.getEnvironment();
OrtSession.SessionOptions opts = new OrtSession.SessionOptions();
OrtSession session = env.createSession("model.onnx", opts);

🌐 Variant 3: TensorFlow Java

Əgər model .h5 formatındadırsa və TensorFlow modelidirsə:

  • TensorFlow Lite və ya TensorFlow Java API istifadə oluna bilər.

Amma PyTorch üçün ən yaxşı yol ya REST API, ya da ONNX ilə inteqrasiyadır.

💻 Kurs Tapşırığı 5️⃣ – Sən nə etməlisən?

  1. Öz .pth modelini ya Flask/FastAPI serverdə yüklə və predict API yaz.
  2. Java ilə bir HTTP sorğu göndərərək bu API-dən nəticəni al.
  3. Nəticəni Java frontend və ya backend-də göstər.
